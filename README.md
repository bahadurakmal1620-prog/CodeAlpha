# LinguaTranslate — CodeAlpha Internship Project

AI-powered Language Translation Tool using MyMemory Translation API.
✅ FREE | ✅ No API Key | ✅ No Server | ✅ No Installation

---

## HOW TO RUN (Step by Step)

### Step 1 — Download the project
Download and unzip the project folder to your computer.

### Step 2 — Open the file
Find the file called:
    index.html

Double-click it to open in your browser.
OR right-click → "Open with" → choose Chrome / Firefox / Edge.

### That's it! The translator works instantly. 🎉

---

## Features
- 40+ languages with flag emojis
- Auto language detection
- Copy to clipboard button ✅
- Text-to-Speech for source and translation 🔊
- Swap languages button 🔄
- Character counter (up to 5,000 chars)
- Ctrl + Enter keyboard shortcut to translate
- Fully responsive (works on mobile too)

---

## API Used
MyMemory Translation API — https://mymemory.translated.net
- 100% Free
- No API Key required
- No signup or credit card needed
- 5,000 words/day free limit
